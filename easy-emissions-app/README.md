# Easy Emissions Marketing Console v1

Static HTML marketing alignment console for Easy Emissions.

## Canon
- Primary goal: get more cars through the doors
- Core message: emissions testing is an interruption; Easy Emissions removes it
- First build: marketing alignment, not check-ins
- Expansion lane: Fleet / B2B

## Deploy on Cloudflare Pages
1. Commit `index.html` to the connected GitHub repository root.
2. In Cloudflare Pages, set build command to blank or `None`.
3. Set output directory to `/`.
4. Deploy.
